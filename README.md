# CooSnoo - THE Customizable Snooze Alarm Clock

When i want to snooze, sometimes i want to snooze for 10 mins, sometimes for 20 mins. I want to be able to choose how long to snoooze at the time my alarm is sounding. I couldnt find any other alam clocks which do this, so vibe coded one that works for me. CooSnoo includes selectable 2, 4 or 6 variable snooze time buttons.
